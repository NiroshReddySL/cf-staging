Teams and business plans include

One-Time Migration

All the files and folder in the source will be migrated to the destination.

Delta Migration

All the existing file changes and the newly added content will be migrated to the destination.

Root Folder permissions

When the folders are migrated, all the permissions on the root folder will be migrated to the destination.

Sub-Folder Permissions

In some instance there might be the extra permissions on sub folders a part of the root level inheritance. These permissions will also be migrated to the destination.

Metadata (Time Stamp)

By default CloudFuze migrates the files/folders with the date of migration in the destination cloud. If this feature is opted, CloudFuze will retain the timestamp in the destination.

Versions

CloudFuze by default migrated the latest file version to the destination. When the versions are opted, CloudFuze migrates all the file versions to the destination cloud.

External Shares

Most of the cloud providers will allow to share the folders to the external team members. CloudFuze will identify these external team members and send the shared link to the destination.

Shared Links

All the cloud providers will allow to create shared links. These links can be created on the folders or sub-folders. CloudFuze identifies these links on the folders and create the links on the destination cloud. A CSV file will be created in the destination with the source folder path , source shared link and destination shared link.

Frequently Asked Questions

1) I don’t represent a company, but I use multi-user (business) clouds. Which category do I belong?

Whether you are an individual or a company, as long as it is a multi-user migration/business cloud, it will fall under the business migration category.

2) I represent a company, but I only need single-user (consumer) cloud migration. What should I do?

Even though you represent a company, if the migration is between two single-user clouds, you can simply subscribe to a personal plan.

3) What should I do if your personal data migration plans are not meeting my migration needs?

You can reach out to us for a custom price quote based on the data size.

4) Does CloudFuze provide discount for non-profit organizations?

Yes, we do.

5) I have more questions.

Contact us via chat, email, or phone. We will answer your questions.
